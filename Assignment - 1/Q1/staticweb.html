HTML Text Formatting Tags

Tag Description Example
<b> Defines bold text. This text is bold
<strong> Defines important text (strong emphasis). This text is strong
<i> Defines italic text. This text is italic
<em> Defines emphasized text (emphasis). This text is emphasized
<mark> Defines marked or highlighted text. This text is marked
<small> Defines smaller text. This text is small
<del> Defines deleted text (strikethrough). This text is deleted
<ins> Defines inserted text (underlined). This text is inserted
<sub> Defines subscript text. H2O
<sup> Defines superscript text. x2
<u> Defines underlined text. This text is underlined
<code> Defines a piece of computer code. This is code
<kbd> Defines keyboard input. Ctrl + C
<var> Defines a variable in programming. x = y + 2
<samp> Defines sample output from a computer program. This is sample output
<pre> Defines preformatted text. 
This is preformatted text
<q> Defines a short quotation. This is a quote
<blockquote> Defines a block quotation.
This is a block quote
<abbr> Defines an abbreviation. WHO
<cite> Defines a citation. This is a citation
<bdo> Overrides the current text direction. This text is in RTL
